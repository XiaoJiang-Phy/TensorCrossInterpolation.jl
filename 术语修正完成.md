### 为什么叫"张量火车"？

Tensor Train（张量火车）这个名称来源于其数学结构：

```
A = G¹ -- G² -- G³ -- ... -- Gᴺ
    |     |     |            |
   车厢1  车厢2  车厢3       车厢N
```

每个"站点张量"（Site Tensor）就像火车的一节车厢，通过"键"（Bond）连接在一起，形成一列"火车"。这个比喻形象地描述了数据结构的特点。
